# Patrick Arthofer Photography Portfolio

Static photography portfolio and automation-driven publishing workflow for presenting curated photography online.

## Overview

This project powers a static photography portfolio site built for GitHub Pages and cloud-hosted image delivery.

It is supported by a local automation pipeline that processes exported images, generates web-ready derivatives, extracts metadata, enriches records with AI-based classification, and prepares searchable gallery data for publishing.

## Key Features

- Static photography portfolio website
- Masonry-style responsive gallery
- Searchable image metadata and AI-generated descriptors
- Automated thumbnail and preview generation
- Structured image catalog for organization and retrieval
- Static-site publishing workflow for GitHub Pages
- Cloud image hosting integration for portfolio assets

## Workflow

1. Images are exported from Lightroom
2. The local automation pipeline generates derivatives and metadata
3. AI classification enriches searchable image records
4. Gallery manifest files are regenerated
5. Static website assets are updated
6. Changes are published to the portfolio site
